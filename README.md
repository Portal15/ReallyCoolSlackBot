# MyCoolSlackBot ✨🤖

MyCoolSlackBot is a small, playful Slack bot built for fun, learning, and automating tiny workflows.

## What it can do
- Respond to commands and messages in Slack with friendly replies 🤝
- Send a cool random dog pictures 🚀
- Send a cool cat fact 🔁
- Easy to extend — add new command handlers in `index.js` 🧩

## Highlights
- Minimal setup — keeps things lightweight and focused ⚡
- Designed for tinkering: add features, test, and iterate fast 🛠️
- Great for learning how Slack bots work and for hacky prototypes 🧪

## Quick Start
1. Install dependencies:

```bash
npm install
```

2. Edit `index.js` with your Slack token and handlers.

3. Run locally:

```bash
node index.js
```

4. Invite the bot to your workspace/channel and try a command!

## Structure
- `index.js` — main entry point and message handlers
- `package.json` — npm metadata and scripts
- `text.txt` / `text.txt.pub` — small data files used by the bot

## Contributing
- Tweak `index.js` to add features.
- Open an issue or send a PR if you want help or want to share improvements.

## Troubleshooting
- If messages don't arrive, check your Slack token and bot permissions 🔐
- Use console logs to see incoming events and errors 🐞
